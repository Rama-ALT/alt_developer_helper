Developer Helper

While developing new addons, the developers tend to edit some technical views to include some additional info, and these edits repeat for each new database.
This module will make your life easier by doing all that work for you!

Features

  • Adds the external id to the technical views. (Models, Groups, Menus, Access Rights, Record Rules, ...etc).

  • Displays extra important data in the technical views (Relation and stored attributes in the views of the fields, Inherited models and records count in the views of the models, ...etc).

  • Extends the limit of the fields list view to the maximum count.

  • Adds a custom filter to the fields view to group by the type automatically.
